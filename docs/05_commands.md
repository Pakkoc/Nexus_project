# Topia Empire - 슬래시 명령어 가이드

## 일반 사용자 명령어

### `/내정보`
자신의 XP, 레벨, 화폐 정보를 확인합니다.

**표시 정보:**
- 현재 레벨 및 XP
- 다음 레벨까지 필요 XP
- 토피/루비 잔액
- 출석 정보

---

### `/출석`
일일 출석 체크를 합니다.

**보상:**
- 기본 출석 보상
- 연속 출석 보너스 (연속 일수에 따라 증가)

**제한:**
- 하루 1회만 가능
- 자정(00:00) 기준 초기화

---

### `/인벤토리`
보유 중인 아이템을 확인하고 사용합니다.

**기능:**
- 보유 아이템 목록 표시
- 역할선택권: 색상 선택/교환 버튼
- 만료일 표시

**역할 교환 시:**
- 기존 색상 역할 제거
- 새 색상 역할 부여
- 고정 역할 유지

---

### `/이체 <대상> <금액> [화폐종류]`
다른 유저에게 화폐를 전송합니다.

**매개변수:**
- `대상`: 전송받을 유저
- `금액`: 전송할 금액
- `화폐종류`: 토피(기본) 또는 루비

**제한:**
- 최소 이체 금액 이상
- 수수료 별도 차감
- 자기 자신에게 이체 불가

---

## 관리자 명령어

### `/지급 <대상> <금액> [화폐종류]`
유저에게 화폐를 지급합니다.

**권한:** 화폐 관리자만 사용 가능

**매개변수:**
- `대상`: 지급받을 유저
- `금액`: 지급할 금액 (음수 가능 = 차감)
- `화폐종류`: 토피(기본) 또는 루비

---

### `/상점설치`
현재 채널에 상점 패널을 설치합니다.

**권한:** 서버 관리자

**패널 구성:**
- 상점 아이템 목록
- 구매 버튼

---

### `/장터설치`
현재 채널에 장터 패널을 설치합니다.

**권한:** 서버 관리자

**패널 구성:**
- 목록 보기 버튼
- 등록하기 버튼
- 내 매물 버튼

---

### `/게임생성`
배팅 게임을 생성합니다.

**권한:** 게임 관리 역할

**입력:**
- A팀 이름
- B팀 이름

**생성되는 패널:**
- A팀 배팅 버튼
- B팀 배팅 버튼
- 배팅 마감 버튼 (관리자용)
- 결과 입력 버튼 (관리자용)
- 경기 취소 버튼 (관리자용)

---

## 상호작용 (버튼/셀렉트)

### 상점 패널
- **구매 버튼**: 아이템 구매 (수량 선택 모달)

### 장터 패널
- **목록 보기**: 현재 판매 중인 아이템 조회
- **등록하기**: 내 아이템 판매 등록
- **내 매물**: 내가 등록한 매물 관리

### 인벤토리
- **색상 선택**: 드롭다운에서 원하는 색상 선택
- **교환하기**: 선택한 색상으로 역할 교환

### 게임센터
- **A/B팀 배팅**: 배팅 금액 입력 모달
- **배팅 마감**: 추가 배팅 차단
- **결과 입력**: 승리팀 선택
- **경기 취소**: 모든 배팅금 환불
