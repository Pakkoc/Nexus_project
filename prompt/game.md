## Game 내전 진행 방식

- 참가비 N 토피 고정 (web에서 설정 가능)
- 배팅: 내전에 참여하느 사람들만 배팅할 수 있게 한다.

How to?
카테고리를 만든다 -> 각 카테고리에 맞는 팀 개수를 넣을 수 있는 창이 나온다. -> 팀명 입력 -> 각 팀원들 배팅

롤체, 베그 같은 경우에는 최후의 생존자가 1명이다.
이것이 1등한테만 돈을 주게 되면은 잘하는 사람은 껴주지 않는 문제 발생

1~4등까지 돈을 주는 방식으로한다.

---

롤 5:5 내전은 A팀(5명), B팀(5명) 있고 A팀이이기면 B님의 돈을 A팀에게 주는 형식이지
즉, A팀 인원 5명은 100토피를 내고 200토피를 가져가는거고 B팀은 0원을 가져가게되는거지

롤토체스같은 경우는 8명이서하고 1. 더블업 (2인 1팀, 4개팀) 2.개인전(1인 1팀, 8개팀)이고 이때는 1~4등에게만 각 퍼센테이지로 돈을 주고 5~8등은 0원을 가져가는거야

근데 이 이외에 다양한 게임을 만들수있도록 인원수, 상금에 대해서 커스텀 가능하게도 하고싶은거야.
